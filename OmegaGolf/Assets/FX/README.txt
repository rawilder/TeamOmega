Glow:

	- Add HoverGlow script to object you want to glow on hover.
	- Adjust transition speed + brightness multiplier in Unity inspector to fit needs
	- For best look, use Self-Illumin/Diffuse shader on object Material

	- Currently doesn't work along with Blinking Script
